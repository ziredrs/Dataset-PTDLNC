# Dataset-PTDLNC
Repository này chứa các bộ dữ liệu phục vụ cho các môn học tại trường

## Danh sách dữ liệu
- `RawDataset.xlsx` – Bộ dữ liệu gốc của e_commerce_shopper_behaviour_and_lifestyle (đề tài ptdlnc)
- `Final_BanhComBaoMinh.xlsx` – Dữ liệu Bánh cốm Bảo Minh (đề tài dma)
- `Final_BanhComPhoXua.xlsx` – Dữ liệu Bánh cốm Phố Xưa (đề tài dma)
- `ToyotaCorolla.csv` – Dataset Toyota Corolla 

## Mục đích
Các dataset này được sử dụng cho:
- Phân tích dữ liệu
- Thống kê mô tả
- Visualization
- Thực hành Python (Pandas, Matplotlib, Seaborn)

## Công cụ sử dụng
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
